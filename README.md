# AssetFlow — Enterprise Asset & Resource Management System

AssetFlow is a modern Enterprise Asset & Resource Management platform developed for the **Odoo Hackathon 2026**. It helps organizations manage assets, streamline maintenance workflows, optimize resource bookings, and support audit compliance through a centralized, role-based system.

## ✨ Features

- 🔐 Secure Authentication & Role-Based Access Control
- 📊 Interactive Dashboard with Analytics
- 🏢 Department & Employee Management
- 📦 Asset Registration & Category Management
- 📅 Resource Booking & Scheduling
- 🛠️ Maintenance Request & Approval Workflow
- ✅ Asset Verification & Audit Management
- 🔔 Notifications & Activity Tracking
- 📈 Reports & Insights

## 🛠️ Tech Stack

**Frontend**
- React.js
- Vite
- React Router
- Tailwind CSS

**Backend**
- Node.js
- Express.js

**Database**
- PostgreSQL (NeonDB)

## 🚀 Getting Started

### 1) Frontend (React)

```bat
cd assetflow-react
npm install
npm run dev
```

### 2) Backend (Node/Express)

```bat
cd backend
npm install
npm run dev
```
